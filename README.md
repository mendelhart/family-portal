# Family Portal

A self-contained family portal: combined Yiddish (Hebrew) + English calendar, holidays, contacts/phone book, events with reminders, and per-family logins.

- **Open `index.html`** to use it. Demo login `admin` / `1234` (local only).
- **`LAUNCH.md`** — step-by-step launch guide.
- **`SETUP.md`** — full setup, reminders, and hosting details.
- **`schema.sql`** — Supabase database + security rules for going live.

Served as a static site (e.g. GitHub Pages). No build step.
